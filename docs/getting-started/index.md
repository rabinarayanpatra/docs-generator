# Getting Started

Welcome to the documentation generator! This guide will help you get up and running quickly.

## Quick Start

Get started with the documentation generator in just a few minutes:

1. **Installation**: Install the required dependencies
2. **Configuration**: Set up your docs configuration
3. **Writing**: Start writing your documentation

Check out the [installation guide](./installation) for detailed instructions.

## Features

- **Markdown & MDX**: Write docs in Markdown or MDX
- **Frontmatter**: Add metadata to your pages
- **Relative Links**: Link between docs easily
- **Code Highlighting**: Beautiful syntax highlighting
- **Dark Mode**: Built-in dark mode support
